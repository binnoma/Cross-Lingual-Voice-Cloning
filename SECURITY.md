# Security Policy | سياسة الأمان

## Reporting Security Vulnerabilities | الإبلاغ عن الثغرات الأمنية

If you discover a security vulnerability, please report it responsibly:

إذا اكتشفت ثغرة أمنية، يرجى الإبلاغ عنها بمسؤولية:

- **Do NOT** open a public GitHub issue
- Send an email to the maintainer: @binnoma
- Include detailed steps to reproduce the vulnerability

## Ethical Use | الاستخدام الأخلاقي

This project is designed for legitimate voice cloning purposes. We actively discourage:

هذا المشروع مصمم لأغراض استنساخ الصوت المشروعة. نحن نثبط:

- ❌ Impersonation without consent | انتحال الشخصية بدون موافقة
- ❌ Creating misleading deepfakes | إنشاء محتوى مضلل
- ❌ Any illegal activities | أي أنشطة غير قانونية

## Security Measures | تدابير الأمان

- Rate limiting on API endpoints
- File size restrictions for uploads
- Audio watermarking (planned)
- Usage monitoring and logging
